# SECTION 3: PRICING STRATEGY

## 3.1 Price Structure

| Platform | Price Model | Amount | Strategy |
|----------|-------------|--------|----------|
| **Flippa** | Auction Starting Bid | $199 | Low entry point se bidding start — attention attract karta hai |
| **Flippa** | Reserve Price | $499 | Isse neeche sell nahi hoga. Automatic reject |
| **Flippa** | Buy Now | $999 | Jo jaldi chahata hai, wo directly le sakta hai |
| **Acquire.com** | Asking Price | $999 | Fixed price ya "OBO" (Or Best Offer) |
| **Acquire.com** | Negotiation Range | $500-$2,000 | Buyer se negotiate kar sakte ho |

## 3.2 Kaise Negotiate Kare

**Rule #1: Kabhi pehle price drop mat karo**
- Buyer puchega: "Can you do $300?"
- Tum reply: "I'm flexible, but $999 is already very fair for a complete SaaS codebase. I can do $849 if you can close today."

**Rule #2: Counter-offer always do**
- Buyer ka offer: $400
- Tumhara counter: $749
- Target: middle ground pe aao — $600-$700 range mein settle karo

**Rule #3: Value show karo, sirf code nahi**
- "This includes 30 days of support, complete documentation, and a ready-to-deploy codebase."
- "You're saving months of development time. A developer would charge $3,000-$5,000 for this."

**Rule #4: Urgency create karo (subtly)**
- "I have another buyer interested. If you can close by [date], I'll do $750."
- — Yeh actually kaam karta hai. Lekin false claim mat karo agar koi interested nahi hai.

**Rule #5: Bundle offers consider karo**
- Agar buyer zyada pay kar raha hai, toh extra support offer karo: "At $900, I'll include 60 days of support instead of 30."
- Extra documentation ya setup assistance bhi offer kar sakte ho

## 3.3 Price Drop Timeline (Agar 7 din mein sell na ho)

| Day | Action |
|-----|--------|
| Day 1-3 | Full price — $999 Buy Now |
| Day 4-5 | Drop to $849 Buy Now |
| Day 6-7 | Drop to $749 Buy Now |

# SECTION 5: SAFE PAYMENT

## 5.1 Escrow Services

| Service | Platform | Safety Level | Fees |
|---------|----------|--------------|------|
| **Escrow.com** | Independent | ⭐⭐⭐⭐⭐ Highest | ~2.89% (buyer usually pays) |
| **Acquire.com Escrow** | Acquire.com | ⭐⭐⭐⭐⭐ Built-in | Included in platform fee |
| **Flippa Escrow** | Flippa | ⭐⭐⭐⭐ Built-in | Included |
| **Direct Transfer** | N/A | ⭐ Very Risky | Free |

**RECOMMENDATION: HAMESHA ESCROW USE KARO. Direct bank transfer ya UPI se kabhi bhi sell mat karo.**

## 5.2 Escrow Kaise Kaam Karta Hai (Step by Step)

```
Step 1: Buyer ne payment Escrow mein deposit kiya ✅
Step 2: Escrow confirm karta hai "Funds Secured" ✅
Step 3: Tum code deliver karte ho (GitHub transfer / download link) ✅
Step 4: Buyer check karta hai (inspection period — usually 3-7 days) ✅
Step 5: Buyer ne accept kar liya → Escrow payment tumhare account mein release karta hai ✅
Step 6: Paise tumhare bank account mein aate hain (2-5 business days) ✅
```

## 5.3 Kaise Ensure Kare Ki Paise Milega

- ✅ **Hamesha platform ke escrow use karo** — Acquire.com, Flippa, ya Escrow.com
- ✅ **Kabhi bhi pehle code mat do** — payment confirm hone ke baad hi deliver karo
- ✅ **Inspection period clearly define karo** — 3-7 days max. Zyada nahi
- ✅ **"As-Is" sale mention karo** — buyer ko pata hona chahiye ki no warranty hai
- ✅ **Delivery confirmation lo** — buyer ko platform pe "Received & Accepted" click karwao
- ✅ **Screenshot lo** — GitHub transfer confirmation, Escrow payment confirmation, buyer ke messages

## 5.4 Fraud Se Kaise Bachega

| Red Flag | Kya Karo |
|----------|----------|
| Buyer kehta hai "Direct UPI/Bank transfer karo, escrow fees bachenge" | **BLOCK.** Hamesha escrow use karo |
| Buyer pehle code maangta hai "review ke liye" | **REFUSE.** Demo link do, code nahi |
| Buyer bahut zyada negotiate karta hai ($999 mein $100 offer) | **IGNORE ya DECLINE.** Time wasters |
| Buyer personal email se contact karta hai, platform ke bahar | **STAY ON PLATFORM.** Platform messages hi use karo for record |
| Buyer delivery confirm nahi karta (inspection period expire hone de) | Escrow auto-release hota hai after inspection period |
| Fake payment proof bhejta hai | Sirf escrow platform ka official confirmation maano, koi screenshot nahi |

**Golden Rule: No code before confirmed payment in escrow. No exceptions.**
